# 🐍 Snake Game (JavaScript)

## 📌 Opis

Gracz steruje wężem, zbiera jedzenie i unika kolizji ze ścianami oraz własnym ciałem.

---

## 🎮 Funkcje

* Sterowanie klawiaturą (strzałki)
* System punktów
* Zapisywanie rekordu (localStorage)
* Pauza (SPACJA)
* Menu startowe z poziomami trudności
* Ekran końca gry (restart / powrót do menu)
* Efekt glow dla węża
* Dynamiczne zwiększanie prędkości (co 5 punktów)

---

## 🕹 Sterowanie

* ⬅️ ➡️ ⬆️ ⬇️ – ruch
* SPACE – pauza

---

## ⚙️ Technologie

* HTML5
* CSS3
* JavaScript (Canvas API)

---

## 🚀 Jak uruchomić

1. Pobierz plik HTML
2. Otwórz w przeglądarce
3. Graj 😎
